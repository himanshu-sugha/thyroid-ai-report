AI in Thyroid Detection Research

A curated list of significant papers, sorted by year.

Year Title & Link Summary
2025 Thyroid Diseases Detection Using Evolutionary Machine Learning and Deep Learning: A survey Surveys ML, DL, and evolutionary algorithms for thyroid diagnosis, concluding that transfer learning and optimization algorithms are most efficient.
2024 Enhanced Diagnosis of Thyroid Diseases Through Advanced Machine Learning Methodologies Develops a classification model for three types of thyroid disease, using SMOTE for class imbalance and achieving 99.76% accuracy with a Gradient Boosting Classifier.
2024 Machine Learning Techniques for the Classification of Thyroid Disease Focuses on classifying thyroid disease into four categories using logistic regression, achieving up to 86% accuracy.
2024 Enhanced interpretable thyroid disease diagnosis by leveraging synthetic oversampling and machine learning models Uses SMOTE-NC for data balancing and a Light Gradient Booster Machine (LGBM) to achieve 96% accuracy, incorporating SHAP for explainability.
2024 Deep learning technology for classification of thyroid nodules using multi-view ultrasound images Evaluates deep learning models (ResNet, DenseNet, EfficientNet) on ultrasound images, finding that better image resolution and refined algorithms improve performance.
2024 Explainable artificial intelligence and machine learning algorithms for classification of thyroid disease Compares various ML models (Random Forest, SVM, XG-Boost) for classifying thyroid disease based on patient clinical data.
2023 Applications of machine and deep learning to thyroid cytology and histopathology: a review Reviews how ML and DL can improve thyroid cancer diagnosis from cytology and histopathology images, addressing issues like interobserver variability.
2023 Artificial Intelligence in Thyroidology: A Narrative Review of the Current Applications, Associated Challenges, and Future Directions Reviews AI applications for thyroid nodules and cancer, highlighting challenges like the need for prospective validation and diverse datasets.
2023 Analysis and classification models used in thyroid disease Surveys various ML classification algorithms for thyroid detection using the UCI dataset, including k-NN, decision trees, and SVMs.
2023 Exploring the Challenges of Diagnosing Thyroid Disease with Imbalanced Data and Machine Learning: A Systematic Literature Review A systematic review of 41 papers analyzing ML/DL approaches for thyroid diagnosis and techniques for handling imbalanced datasets.
2022 A deep learning framework for classifying common vs. rare thyroid cancer subtypes from whole-slide images Presents a deep learning framework using a UNet model to classify thyroid cancer subtypes from whole-slide images, achieving a high AUC.
2022 A Novel Technique for Detecting Various Thyroid Diseases Using Deep Learning Discusses using a CNN with a modified ResNet architecture to detect five different types of thyroid disease.
2022 A novel machine learning-based thyroid disease prediction approach Proposes a multi-class prediction approach using an extra tree classifier for feature selection with a random forest, achieving 99% accuracy.
2021 Machine-Learning-Based Multiple-Instance Learning Model for Thyroid Cancer Detection Using Ultrasound Images Compares SVM and ANN algorithms for classifying thyroid nodules, finding that the SVM model performed better on all metrics.
2020 Classification of Thyroid Nodules with Stacked Denoising Sparse Autoencoder Developed a classifier using immune-related genes and a stacked denoising sparse autoencoder, achieving 92.9% accuracy in discriminating malignant and benign nodules.
2020 Computer-aided diagnosis systems in diagnosing malignant thyroid nodules: a systematic review and meta-analysis A meta-analysis finding that both classic ML and deep learning-based CAD systems showed good performance, comparable to radiologists.
2019 Deep convolutional neural networks for differentiating thyroid nodules Explored using DCNNs (specifically VGG-19) on histopathological slides, yielding a diagnostic accuracy of 97.34%.
2018 Machine Learning Techniques for Thyroid Disease Diagnosis: A Systematic Review A systematic review of various classification and dimensionality reduction algorithms used in ML for diagnosing thyroid disease.
2016 A comparative evaluation of unsupervised anomaly detection algorithms for multivariate data Modified the original UCI Thyroid Disease Data Set for unsupervised anomaly detection, creating the widely used "annthyroid" dataset.
2011 SMOTE: Synthetic Minority Over-sampling Technique A foundational paper describing the SMOTE technique for over-sampling the minority class to address imbalanced datasets.

Note: This is a curated, non-exhaustive list for informational purposes. Links direct to external academic sites.
